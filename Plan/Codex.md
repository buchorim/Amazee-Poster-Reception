# Codex — Amazee SNBT Style Poster

## What Is This
Amazee SNBT Style Poster is a local poster asset project containing vertical Amazee result images. The project organizes poster images so they can be identified, shared, and delivered without relying on generated numeric filenames.

## Who Is It For
This project is for the owner managing Amazee poster assets and needing clean, participant-based poster delivery links.

## Business Model
Internal tooling - no monetization.

## Pricing
Not applicable - internal asset project.

## How to Use
Open the project folder, go to the `ASSET` directory, and select the poster image by participant name. For sharing, use the QR images in `Qr` or the name-only public links in `Links.md`.

## Key Constraints & Assumptions
Only filenames and delivery metadata are changed during asset organization unless the owner explicitly asks for image editing. Poster image pixels, dimensions, and file formats stay unchanged. Public URLs must not reveal outcome metadata before the poster opens.
