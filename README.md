# Amazee SNBT Style Poster

Static GitHub Pages poster viewer for Amazee poster links.

## GitHub Pages Setup

1. Push this repository to GitHub.
2. Open repository settings.
3. Go to Pages.
4. Set source to deploy from the main branch and repository root.
5. Use the published GitHub Pages URL as the base URL for poster links.

## Poster Link Format

```txt
<GitHubPagesBaseUrl>/?p=<name-slug>
```

Published base URL:

```txt
https://buchorim.github.io/Amazee-Poster-Reception/
```

Example:

```txt
https://buchorim.github.io/Amazee-Poster-Reception/?p=akbar-vijaya
```

The full slug list is in `Links.md`.
